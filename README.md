# 🧮 Tkinter Calculator

A **simple calculator** built with Python's Tkinter library.  
It performs basic arithmetic operations: addition, subtraction, multiplication, division, percentage, and decimal points.  
The interface is clean, responsive, and features a dark colour scheme.

## ✨ Features

- Basic operations: `+`, `-`, `*`, `/`, `%`
- Clear (`C`) button to reset the input
- Decimal (`.`) support
- Error handling for invalid expressions (displays "error")
- Fixed window size (570×600) – no resizing
- Dark theme with colourful buttons
